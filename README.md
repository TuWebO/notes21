# 21notes

A Python project for music notation and analysis.

## Setup

1.  Create and activate a virtual environment:
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    ```

2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

To run the main application:
```bash
python main.py
```

## Testing

To run tests:
```bash
pytest
```
